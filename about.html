Installation

To install and shader pack it is a simple and quick process. follow these steps to install your first shader pack

1. Download a shader pack from the mods section or search up a shader pack

2. Navigate to the .zip folder in your downloads folder or where ever you saved it to

3. In a seperate file explorer window navigate to the .minecraft folder which is located in "C:\Users\(your name)\AppData\Roaming\.minecraft"

4. Find the shader packs folder which is labeled shaderpacks and drag the .zip shader folder into the shader packs folder

5. Open Minecraft and enjoy your new shaders
